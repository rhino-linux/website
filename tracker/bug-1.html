    Rhino Linux | Pacstall update bug

    While all of our Pacscripts have been updated to beta5, there is an issue with Pacstall that breaks certain Pacscript updates. While we work to resolve this issue and Plasma releases a new version of Pacstall, you can feel free to upgrade to new releases with pacstall -PI rhino-core linux-kernel-stable and then afterwards run rpk update to recieve the latest changes.
