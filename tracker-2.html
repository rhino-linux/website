Bug #2 (RESOLVED)

A minor issue with the setup wizard currently is that the setup wizard does not install snapd. It still installs support for Flatpaks and AppImages, however it does not install or configure Snap. The workaround is once you have gone through the setup, to manually install snapd on your system before rebooting. You can install snapd with sudo apt install snapd -y
