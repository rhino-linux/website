Packaging UBXI

Reference

While the details of the implementation of an UBXI DE are directly tied to it's base DE, the pacscript for unicorn-desktop-git should be used as a reference for UBXI packages. The full Pascript packaging guide can be found on the Pacstall Wiki.

Dotfiles

All of the dotfiles needed for the configuration of an UBXI DE must be placed in a git source repository, preferably on GitHub or GitLab, similar to how the Unicorn repository is structured.

Packaging standards

All UBXI packages must follow the UBXI naming convention. That is ubxi-{environment}-{platform}-git, where environment is the desktop environment/window manager base that it is built ontop of, and platform is either desktop or mobile.

All UBXI packages are to be submitted to The Pacstall Programs repository for testing and review by both the Pacstall and Rhino Linux teams before approval.
