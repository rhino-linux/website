    Rhino Linux | Pacstall update bug

    While all of our Pacscripts have been updated to beta5, the issue with Pacstall is that it currently does not have a dependency tree, and so some packages get installed before their dependencies causing the installation of the package to fail. While we work to resolve this issue and Plasma releases a new version of Pacstall, you can feel free to upgrade to new releases with pacstall -PI rhino-core linux-kernel-stable and then afterwards run rpk update to recieve the latest changes.
