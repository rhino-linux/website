Bug #5

Note: this issue only occurs if you are upgrading from a beta release to the full release of Rhino Linux. If you did a fresh installation of Rhino Linux 2023.1 or higher then this will not affect your system.

With Pacstall upgrading to v4.0.0 some users on 2023.1-beta7 will not get updates to the Unicorn Desktop or rhino-pkg. This can be fixed really easily by running:


pacstall -PI unicorn-desktop-git rhino-pkg-git

Or if you are using Pine64 devices:


pacstall -PI unicorn-mobile-git rhino-pkg-git
