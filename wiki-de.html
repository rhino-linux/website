How to install other desktop environments

Some users may not be happy with the Unicorn Desktop that they are provided, and may want to change their desktop environment to one they are more comfortable with, such as GNOME or KDE Plasma.

First ensure your software repositories are up to date with:

rhino-pkg update -y

Next you can remove the Unicorn Desktop environment safely with:

pacstall -PI rhino-server-core

Running this ensures you do not remove any Rhino Linux dependencies, but instead remove our desktop environment and graphical applications.

After this, you can now install any desktop environment you like from the package names in the Ubuntu repositories.
